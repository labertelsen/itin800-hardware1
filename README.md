# itin800-hardware1

Arduino and hardware programming projects completed for ITIN 8000 Hardware 1.

## Overview

This repository includes embedded systems exercises using Arduino-compatible hardware, sensors, and actuators. Projects explore basic hardware control, environmental sensing, and interactive electronics.

## Technologies

Arduino, C/C++, Sensors, Microcontrollers

## Skills Demonstrated

* Embedded systems programming
* Sensor integration
* Hardware prototyping
* Arduino-based experimentation
